# Monitoria-Introducao-a-Algoritmos

1. **Variáveis e Tipos de Dados:**
    ```python
       Nome:Lucas
       idade = 18
       peso = 62.5
       print(f"Nome: {nome} Idade: {idade} Peso: {peso:.1f}Kg")
    ```
    **Int:** Um dado do tipo Int como a idade só aceita **numeros inteiros**, ou seja, números sem vírgula positivos e negativos. Ex: idade
    **Float:** Um dado do tipo Float é capaz de guardar números fracionários (com vírgula). Ex: temperatura, peso, altura.
    **String:** Um dado do tipo String armazena um texto e deve ser escrito entre aspas.
    **Declaração de Variáveis**
       Para declarar uma variável é necessário atribuir um nome a ela, colocar o símbolo de igual e depois o seu valor ou conteúdo(se for uma string).
   **A função Print**
        O print serve para exibir uma mensagem na tela. É possível exibir variáveis na tela! Como? escrevendo:
   ```python
       print(f"{nome da variavel}
   ```
   Para exibir dados Float deve-se fazer o seguinte:
   ```python
       print(f"{nome da variavel:.1f}
   ```
   O que serve para exibir um número do tipo float com apenas uma casa decimal, caso contrário ele exibirá várias casas na tela.
